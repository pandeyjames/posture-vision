About

Posture Vision

Local seated posture monitoring for forward lean, breaks, absence checks, and optional habit insights.

Privacy disclaimer

Posture Vision does not record video, does not record audio, and does not save screenshots or webcam images. Webcam frames are processed locally for posture landmarks only.

Prerequisites Posture Vision needs Windows 10/11, a modern browser, webcam permission, and internet access for first install/model loading. The installer can install Python automatically if it is missing.
Background operation Use the Desktop shortcut or tray Open app command. It opens a standalone app window and keeps a Posture Vision tray icon available. Minimize to tray hides the window while monitoring continues.
Saved locally Calibration, settings, consent, statistics, profiles, and optional habit logs are saved in the local SQLite database.
Not medical advice This app is a posture reminder and productivity aid. It is not a medical, diagnostic, or ergonomic assessment tool.
Model files The browser loads MediaPipe runtime/model files from external CDNs unless the app is packaged with bundled assets later.

Local posture and habit monitoring

Posture Vision uses your webcam to estimate seated posture. Processing runs locally in this browser.

The app does not record video, audio, screenshots, or webcam images. Calibration, statistics, and optional habit logs are stored locally on this computer.

Habit and expression-signal features are optional and describe visible patterns only. They do not determine actual emotions or health conditions.
